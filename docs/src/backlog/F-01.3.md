# [FRONTEND] F-01.3 — Añadir Stack Tecnológico de `<Nombre>`

## 🏷️ Capa Técnica
**Capa:** `Frontend / Markdown`
**Impacta en:** `docs/team.md`
**Prioridad:** `Must`

## Descripción

Como miembro del equipo, quiero publicar las tecnologías que conozco o estoy aprendiendo en `team.md`, usando etiquetas visuales (`badges`), para que el equipo identifique rápidamente quién puede ayudar con qué stack.

## 📦 Bloque a insertar

Añade el siguiente bloque en `team.md` dentro del perfil del miembro, sustituyendo los badges por los tuyos. Puedes generar badges en [shields.io](https://shields.io) o usar los ejemplos de abajo:

```md
#### 🛠️ Stack Tecnológico — <Tu Nombre>

**Lenguajes:**
![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)
<!-- Añade o elimina los que correspondan -->

**Frameworks / Librerías:**
![<Nombre>](https://img.shields.io/badge/<Nombre>-<Color>?style=flat&logo=<logo>&logoColor=white)

**Herramientas:**
![Git](https://img.shields.io/badge/Git-F05032?style=flat&logo=git&logoColor=white)
![VSCode](https://img.shields.io/badge/VSCode-007ACC?style=flat&logo=visual-studio-code&logoColor=white)
<!-- Añade las tuyas -->
```

> 💡 **Mínimo requerido:** al menos **3 tecnologías** reales con su badge correspondiente.

## ✅ Criterios de Aceptación (Checklist)

- [ ] **Escenario 1:** El bloque contiene un mínimo de 3 badges de tecnologías reales que el miembro conoce o está aprendiendo.
- [ ] **Escenario 2:** Los badges se renderizan como imagen en Vitepress (no aparece la URL en texto plano).
- [ ] **Escenario 3:** No hay placeholders `< >` sin sustituir en el documento final.
- [ ] **Requisito Técnico (UserManual):** Se ha añadido una sección `## UserManual` explicando cómo se generaron los badges y cómo otro miembro podría añadir los suyos.

```bash
git checkout -b feature/F-01.3-stack-<tu-alias>
```
