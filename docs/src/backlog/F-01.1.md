# [FRONTEND] F-01.1 — Añadir Tarjeta de Presentación Básica de `<Nombre>`

## 🏷️ Capa Técnica
**Capa:** `Frontend / Markdown`
**Impacta en:** `docs/team.md`
**Prioridad:** `Must`

## Descripción

Como miembro del equipo, quiero añadir mi tarjeta de presentación básica en `team.md` para que el resto del equipo sepa quién soy, qué estudio y cuál es mi rol en el proyecto.

## 📦 Bloque a insertar

Copia el siguiente bloque en `docs/team.md`, dentro de la sección `## 👥 Miembros del Equipo`, y **sustituye todos los valores entre `< >`** por tu información real:

```md
### 🧑‍💻 <Tu Nombre Completo>

| Campo      | Valor                          |
|------------|-------------------------------|
| **Nombre** | `<Tu Nombre Completo>`        |
| **Alias**  | `<Tu nombre en GitHub>`       |
| **Curso**  | `<DAW / DAM / ASIR / ...>`    |
| **Rol**    | `<Frontend / Backend / DevOps / Full Stack>` |
| **Grupo**  | `<Número de grupo>`           |
```

## ✅ Criterios de Aceptación (Checklist)

- [ ] **Escenario 1:** La tarjeta aparece dentro de la sección `## 👥 Miembros del Equipo` de `team.md`.
- [ ] **Escenario 2:** Todos los campos con `< >` han sido reemplazados por datos reales.
- [ ] **Escenario 3:** La tabla Markdown se renderiza correctamente en Vitepress (sin columnas rotas).
- [ ] **Requisito Técnico (UserManual):** Se ha añadido una sección `## UserManual` explicando qué bloque se insertó y cómo replicarlo.
