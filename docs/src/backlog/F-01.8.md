# [FRONTEND] F-01.8 — Añadir Recomendación Cultural (Serie, Película o Libro) de `<Nombre>`

## 🏷️ Capa Técnica
**Capa:** `Frontend / Markdown`
**Impacta en:** `docs/team.md`
**Prioridad:** `Should`

## Descripción

Como miembro del equipo, quiero recomendar una serie, película o libro que me haya marcado, añadiéndolo en una sección dedicada en `team.md`, para enriquecer la página con contenido personal y fomentar el intercambio de recomendaciones culturales entre el equipo.

## 📦 Bloque a insertar

Añade el siguiente bloque en tu sección de `team.md`, sustituyendo los valores entre `< >`:

```md
#### 🎬 Recomendación Cultural — <Tu Nombre>

| Campo         | Valor                                          |
|---------------|------------------------------------------------|
| **Tipo**      | `Serie` / `Película` / `Libro` *(elige uno)*   |
| **Título**    | *<Título de la obra>*                          |
| **Género**    | `<Drama / Sci-Fi / Terror / Comedia / ...>`    |
| **¿Por qué?** | <Una o dos frases explicando qué te aportó>   |
| **Puntuación**| `⭐⭐⭐⭐⭐` *(de 1 a 5 estrellas)*               |
```

> 💡 Solo puedes recomendar **una única obra**. Elige la que más impacto haya tenido en ti.

## ✅ Criterios de Aceptación (Checklist)

- [ ] **Escenario 1:** Solo se recomienda una obra (no se listan varias).
- [ ] **Escenario 2:** El campo `¿Por qué?` contiene una justificación real de al menos una frase.
- [ ] **Escenario 3:** La puntuación usa el sistema de estrellas `⭐` del bloque (1 a 5).
- [ ] **Requisito Técnico (UserManual):** Se ha añadido una sección `## UserManual` describiendo el bloque y cómo otro miembro podría añadir su recomendación.
