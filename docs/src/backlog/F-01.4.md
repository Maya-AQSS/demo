# [FRONTEND] F-01.4 — Añadir Datos de Contacto Profesional de `<Nombre>`

## 🏷️ Capa Técnica
**Capa:** `Frontend / Markdown`
**Impacta en:** `docs/contact.md`
**Prioridad:** `Must`

## Descripción

Como miembro del equipo, quiero añadir mis datos de contacto profesional en `contact.md` para facilitar la comunicación interna del equipo y practicar la edición colaborativa de una misma página.

## 📦 Bloque a insertar

Añade el siguiente bloque en `docs/contact.md`, dentro de la sección `## 📇 Directorio del Equipo`, sustituyendo los valores entre `< >` por los tuyos:

```md
### 📬 <Tu Nombre Completo>

| Medio          | Enlace / Valor                                      |
|----------------|-----------------------------------------------------|
| **GitHub**     | [@<tu-usuario>](https://github.com/<tu-usuario>)   |
| **LinkedIn**   | [Ver perfil](https://linkedin.com/in/<tu-perfil>)  |
| **Discord**    | `<TuNombre>#<0000>` *(servidor del equipo)*         |
| **Email inst.**| `<usuario>@<dominio-institucional>`                 |
```

> ⚠️ **No incluyas** número de teléfono personal ni email privado. Solo datos propios de entornos profesionales/académicos.

## ✅ Criterios de Aceptación (Checklist)

- [ ] **Escenario 1:** La fila del miembro aparece dentro de la sección `## 📇 Directorio del Equipo` de `contact.md`.
- [ ] **Escenario 2:** El enlace de GitHub apunta a un perfil real y válido.
- [ ] **Escenario 3:** No se ha incluido información sensible (teléfono personal, contraseñas, DNI…).
- [ ] **Requisito Técnico (UserManual):** Se ha añadido una sección `## UserManual` indicando qué datos se añadieron y cómo otro miembro podría replicarlo.
